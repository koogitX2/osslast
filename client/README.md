# 🥗 학생 식단 추천 사이트 (OSS 최종 과제)

사용자의 키, 몸무게, 영양 요구량을 기반으로 교내 식당에서 최적의 식단 조합을 추천해주는 React 기반 웹 애플리케이션입니다.

## 🚀 최신 업데이트 (v2.0)
-   **MockAPI 연동**: 완전한 서버리스(Serverless) 아키텍처를 구현했습니다. 모든 메뉴 데이터(오늘의 메뉴 포함)를 MockAPI에서 실시간으로 가져옵니다.
-   **2끼 추천 기능 추가**: 아침을 거르는 분들을 위해 "점심 + 저녁" 추천 옵션을 추가했습니다.
-   **상세 영양소 분석**: 칼로리뿐만 아니라 **탄수화물, 단백질, 지방**의 목표량 vs 실제 섭취량을 상세하게 비교 분석해줍니다.

## 🛠 사용 기술 (Tech Stack)
-   **Frontend**: React (Create-React-App 구조)
-   **Data**: MockAPI (Serverless Architecture)
-   **Design**: Custom CSS ("말랑" 디자인 테마)

## 📂 프로젝트 구조

## 🏃‍♂️ 실행 방법 (How to Run)
1.  클라이언트 디렉토리로 이동:
    ```bash
    cd client
    ```
2.  의존성 패키지 설치 (최초 1회):
    ```bash
    npm install
    ```
3.  개발 서버 실행:
    ```bash
    npm start
    ```
4.  브라우저에서 `http://localhost:3000` 접속.

## ✨ 주요 기능
1.  **TDEE(기초대사량) 자동 계산**: 키/몸무게를 입력하면 Mifflin-St Jeor 공식으로 하루 권장 칼로리를 계산합니다.
2.  **스마트 식단 추천**: 목표 칼로리에 가장 근접한 식단 조합(3끼 또는 2끼)을 찾아냅니다.
3.  **영양소 분석 시각화**: 나에게 필요한 영양소와 실제 식단의 영양소를 한눈에 비교할 수 있습니다.
4.  **오늘의 메뉴 (Carousel)**: 교내 다양한 식당들의 오늘 메뉴를 모아서 보여줍니다. (좌우 슬라이드)
5.  **캠퍼스 지도**: 식당 위치를 확인할 수 있는 캠퍼스 지도를 제공합니다.
